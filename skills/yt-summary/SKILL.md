# YouTube 影片重點整理

將影片字幕分析整理成結構化的重點摘要報告並產出 `summary.md`

# 輸出位置

- 該歌曲所在的資料夾

## 輸入

- SRT 字幕檔

## 輸出格式

Markdown 報告，包含：
- 影片標題
- 分段重點（每段包含標題、時間範圍、摘要）
- GIF 動圖佔位

## Markdown 結構範例

```markdown
# 影片重點整理：《影片標題》

## 段落一：段落標題（00:00 - 00:30）
![highlight](./gifs/001.gif)
> 這段主要在講...

## 段落二：段落標題（02:30 - 03:30）
![highlight](./gifs/002.gif)
> 這段主要在講...

## 段落三：段落標題（05:10 - 08:45）
![highlight](./gifs/003.gif)
> 這段主要在講...
```

## 工作流程

1. 讀取 SRT 字幕檔
2. 分析內容，識別重點段落
3. 為每個段落：
   - 撰寫簡潔的標題
   - 標記時間範圍（起始-結束）
   - 撰寫 2-3 句重點摘要
4. 產生 Markdown 報告
5. 標記需要產生的 GIF

## 注意事項

- 時間格式統一使用 `MM:SS` 或 `HH:MM:SS`
- 摘要要精簡扼要，避免過度詳細
