# 工作流程

讀取 SRT 字幕檔，分析內容產出 `summary.md` 到影片所在資料夾。每段包含標題、時間範圍、2-3 句摘要。

若後續需要搭配 `gif` skill 產生動圖，每段可加上 GIF 佔位圖片連結。

## 輸出範例

```markdown
# 影片重點整理：《影片標題》

## 段落一：段落標題（00:00 - 02:30）
> 這段主要在講...

## 段落二：段落標題（02:30 - 05:10）
> 這段主要在講...
```
